The master formula

A = U Σ VT
U (m×m)
Left singular vectors. Columns = ONB for column space + left null space. Rotation in output space.
Σ (m×n)
Diagonal (rectangle). Entries σ₁≥σ₂≥…≥0. Number of nonzeros = rank(A). Pure scaling.
V (n×n)
Right singular vectors. Columns = ONB for row space + null space. Rotation in input space.

Diagonalization & Eigendecomposition

Core idea

For a square matrix A, if we can find n linearly independent eigenvectors, we can write:

A = P Λ P−1

where P has eigenvectors as columns and Λ is diagonal with eigenvalues.

This is the same as writing AP = PΛ, which means each column: Avᵢ = λᵢvᵢ

Condition for diagonalizability

An n×n matrix is diagonalizable if and only if it has n linearly independent eigenvectors.

Key check: For each eigenvalue, the geometric multiplicity (dim of eigenspace) must equal its arithmetic multiplicity (how many times it's a root of the characteristic polynomial). Symmetric matrices always satisfy this.

Example — NOT diagonalizable:

A = [2 1/0 2] has eigenvalue λ=2 repeated twice, but only ONE eigenvector [1,0]ᵀ

Only 1 linearly independent eigenvector → cannot form P with 2 columns → NOT diagonalizable.

Finding eigenvalues

Solve the characteristic polynomial: det(A − λI) = 0

For triangular matrices, eigenvalues are just the diagonal entries. Shortcut for 2×2: λ² − tr(A)λ + det(A) = 0

For upper/lower triangular A = [a b/0 d], eigenvalues are directly a and d.

Finding eigenvectors

For each eigenvalue λᵢ, solve: (A − λᵢI)x = 0

This is just finding the null space of (A − λᵢI). Free variables give you the eigenvectors.

Geometric multiplicity = number of free variables = number of linearly independent eigenvectors for that λ. This must be ≥ 1 and ≤ arithmetic multiplicity.

Building the decomposition

Arrange eigenvectors as columns of P — the ORDER must match eigenvalues in Λ:

P = [v₁ | v₂ | … | vₙ]    Λ = diag(λ₁, λ₂, …, λₙ)

Verify: AP = PΛ. Then A = PΛP⁻¹.

For symmetric matrices: eigenvectors are orthogonal, so P⁻¹ = Pᵀ. We get A = PΛPᵀ (spectral theorem).

Why diagonalization matters: computing powers

Ak = P Λk P−1

Since Λ is diagonal, Λk just raises each diagonal entry to the power k. This is O(n) work per power instead of O(n³) matrix multiplication.

This is why SVD also makes rectangle matrix powers easy — Σk is trivial to compute.

Similarity — the bridge to SVD

B is similar to A if B = QAQ⁻¹ for some invertible Q. Similar matrices represent the same linear transformation in different bases.

Key fact: AB and BA share all nonzero eigenvalues. This is the bridge that lets us build AᵀA and AAᵀ to find SVD.

Gram–Schmidt Orthogonalization

Why orthonormal bases?

If U = [u₁|u₂|…|uₙ] is orthonormal and we want coordinates of x in this basis:

αᵢ = uᵢTx

Just dot products — no matrix inversion! This is why SVD's U and V being orthonormal is so powerful.

Normalize
Extend

Given basis {v₁, v₂, …, vₙ}, we want to produce an orthonormal basis {u₁, u₂, …, uₙ}.

Start: u₁ = v₁ / ‖v₁‖ (just normalize the first vector)

For u₂: take v₂ and subtract the component along u₁:

u₂ = v₂ − (v₂·u₁/‖u₁‖²) u₁

The subtracted piece is the projection of v₂ onto u₁. What remains is perpendicular to u₁.

After subtracting projections, normalize: divide by the length.

u₂ ← u₂ / ‖u₂‖

Orthonormal check: uᵢ · uⱼ = 0 for i≠j (orthogonal), and ‖uᵢ‖ = 1 (unit length).

For u₃, subtract projections onto ALL previous u's:

u₃ = v₃ − (v₃·u₁)u₁ − (v₃·u₂)u₂   then normalize

General pattern:

uₖ = vₖ − Σᵢ₌₁ᵏ⁻¹ (vₖ·uᵢ) uᵢ   then normalize

This is needed in SVD when Σ has zeros — you find remaining U columns by orthogonalizing the null space of Aᵀ.

SVD — Singular Value Decomposition

Unconditional: SVD works for ANY m×n matrix — no symmetry, no square requirement. This is the key advantage over eigendecomposition.

Find V — right singular vectors

Compute AᵀA (n×n, always symmetric positive semidefinite). Find its eigenvectors.

AᵀA = V Σ² Vᵀ

Columns of V = eigenvectors of AᵀA. Since AᵀA is symmetric, these are automatically orthogonal!

AᵀA is always symmetric → eigenvectors always orthogonal → V is automatically an orthogonal matrix.

Find Σ — singular values

From the eigenvalues λᵢ of AᵀA (or AAᵀ), take positive square roots:

σᵢ = √λᵢ

Always positive! Arrange in decreasing order: σ₁ ≥ σ₂ ≥ … ≥ σᵣ > 0, then zeros.

Key identity: singular values of A = √(eigenvalues of AᵀA). Rank(A) = number of nonzero σᵢ.

Singular values of A⁻¹ = 1/σᵢ (just reciprocals!)

Find U — left singular vectors

Compute AAᵀ (m×m, always symmetric). Find its eigenvectors.

AAᵀ = U Σ² Uᵀ

Columns of U = eigenvectors of AAᵀ.

AᵀA and AAᵀ share the same nonzero eigenvalues (AB and BA theorem). So the nonzero singular values come out the same either way.

Shortcut: compute U from V without AAᵀ

Since Avᵢ = σᵢuᵢ, once you have V and Σ you can get each uᵢ directly:

uᵢ = (1/σᵢ) A vᵢ

This avoids computing the full AAᵀ and is the standard computational approach.

Verify: check that uᵢ · uⱼ = 0 for i≠j. They will be orthogonal because AᵀA is symmetric.

Finding remaining U columns (when σᵢ = 0)

You can only use the shortcut for nonzero σᵢ. For the remaining r+1 … m columns of U:

  1. Find null space of Aᵀ (solve Aᵀx = 0)
  2. Apply Gram–Schmidt to get orthonormal basis
  3. These become the remaining U columns

If λ=0 is repeated for AᵀA, the eigenvectors may not be automatically orthogonal — you MUST apply Gram–Schmidt to get the orthonormal set.

Geometry of SVD

Three operations: Vᵀ → Σ → U

① Vᵀ — rotate
Orthonormal matrix → pure rotation. Preserves lengths and angles. Maps vᵢ → eᵢ (standard basis).
② Σ — stretch
Diagonal matrix → pure scaling. Stretches/shrinks each axis by σᵢ. Circle → ellipse.
③ U — rotate
Another orthonormal matrix → pure rotation in output space. Orients the stretched ellipse.

Every matrix = rotate input → stretch → rotate output. The singular values tell you exactly how much stretching happens.

Length bounds after transformation

For any unit vector w, after multiplying by A = UΣVᵀ:

σₘᵢₙ ≤ ‖Aw‖ ≤ σₘₐₓ

The smallest singular value gives the minimum stretch, the largest gives the maximum stretch.

  • U and Vᵀ are orthonormal → length-preserving (no stretching)
  • Only Σ changes lengths — by exactly σᵢ along axis i
  • ‖Aw‖ = σ₁ only if w is aligned with v₁ (the first right singular vector)

Orthonormal matrices: length and angle preserving

Length preserving

‖Ux‖² = (Ux)ᵀ(Ux) = xᵀUᵀUx = xᵀIx = ‖x‖²

UᵀU = I → ‖Ux‖ = ‖x‖ always.

Angle preserving

Ux · Uy = (Ux)ᵀ(Uy) = xᵀUᵀUy = xᵀy = x · y

Dot products preserved → angles preserved.

Four Fundamental Subspaces via SVD

SVD directly reveals all four fundamental subspaces of A. If rank(A) = r, the SVD A = UΣVᵀ gives you all four bases at once.

Row space of A

Span{v₁,…,vᵣ} — first r right singular vectors. dim = rank(A) = r. These are the vectors A maps to nonzero outputs.

Null space of A

Span{vᵣ₊₁,…,vₙ} — last n−r right singular vectors. dim = n−r. Avⱼ = 0 for j > r.

Column space of A

Span{u₁,…,uᵣ} — first r left singular vectors. dim = rank(A) = r. These are exactly the outputs Av can produce.

Left null space of A

Span{uᵣ₊₁,…,uₘ} — last m−r left singular vectors. dim = m−r. These are perpendicular to all outputs.

The key equations connecting SVD and subspaces

From A = UΣVᵀ, multiply on the right by V (V is orthogonal, so VᵀV = I):

AV = UΣ

Column by column this gives:

Avᵢ = σᵢuᵢ for i = 1,…,r  |  Avⱼ = 0 for j = r+1,…,n

This single family of equations defines all four subspaces simultaneously.

Rank-nullity from SVD: rank + nullity = n. We have r nonzero σ → rank = r → nullity = n−r. Directly readable from Σ.

Null space proof: rank(A) = rank(AᵀA)

We need to show null(A) = null(AᵀA):

  • If Ax = 0 → then AᵀAx = Aᵀ(0) = 0. So null(A) ⊆ null(AᵀA).
  • If AᵀAx = 0 → multiply xᵀ on left: xᵀAᵀAx = 0 → ‖Ax‖² = 0 → Ax = 0. So null(AᵀA) ⊆ null(A).

Therefore null(A) = null(AᵀA) → same nullity → same rank. ✓

SVD as Sum of Rank-1 Matrices

Column × Row decomposition

Instead of row × column (the usual way), we can multiply column × row to get a rank-1 matrix. SVD makes this beautiful:

A = σ₁u₁v₁ᵀ + σ₂u₂v₂ᵀ + … + σᵣuᵣvᵣᵀ

Each term σᵢuᵢvᵢᵀ is a rank-1 matrix. A is the sum of r such matrices.

Energy captured = Σᵢ₌₁ᵏ σᵢ² / Σⱼ σⱼ². Drop small σᵢ → compress data with minimal loss.

Applications of rank-1 decomposition

Image compression

Keep top-k singular values → approximate the image. Storage: k(m+n+1) numbers instead of mn.

PCA / Dimensionality reduction

Top-k right singular vectors = principal components. Project data onto them to reduce dimensions.

Condition number

κ(A) = σₘₐₓ/σₘᵢₙ. Large condition number → near-singular → numerically unstable.

Best rank-k approximation

Eckart–Young theorem: the truncated SVD gives the best rank-k approximation in both 2-norm and Frobenius norm.

True / False — common exam traps

FALSE
"Singular values of A² = (singular values of A)²"
Counterexample: A = [[0,1],[0,0]], σ(A)={1,0} but A²=0, σ(A²)={0,0}.
FALSE
"For square A, singular values = |eigenvalues|"
Same counterexample: eigenvalues of [[0,1],[0,0]] are both 0, but singular value is 1.
FALSE
"Rank = number of nonzero eigenvalues (for any matrix)"
True only for symmetric matrices. For general matrices use SVD (nonzero singular values).
TRUE
"Every real matrix has an SVD"
Unconditional — no conditions on A.
TRUE
"Singular values of Q are all 1 (Q orthogonal)"
Orthonormal matrix doesn't scale → all σᵢ = 1.
TRUE
"Singular values of A⁻¹ = 1/σᵢ"
If A = UΣVᵀ then A⁻¹ = VΣ⁻¹Uᵀ, so singular values are 1/σᵢ.
TRUE
"If B similar to A and A diagonalizable → B diagonalizable"
Similarity is transitive. B = QAQ⁻¹ = Q(PΛP⁻¹)Q⁻¹ = (QP)Λ(QP)⁻¹.

Quiz — test yourself

Click an option to reveal the answer. Each question is based on the lecture material.

Q1. Matrix A is 3×5 with rank 2. What is the dimension of its null space?

Null space of A is in ℝ⁵ (the input space). By rank-nullity: nullity = n − rank = 5 − 2 = 3.

Q2. Given SVD A = UΣVᵀ, which vectors form a basis for the column space of A?

Column space = span of left singular vectors u₁,…,uᵣ = first r columns of U. These satisfy Avᵢ = σᵢuᵢ, so they ARE in the column space.

Q3. An orthonormal matrix Q has all singular values equal to…

Q is orthonormal → QᵀQ = I → eigenvalues of QᵀQ are all 1 → singular values = √1 = 1. Geometrically: orthonormal matrices only rotate, never stretch.

Q4. For the Gram–Schmidt process, what does subtracting the projection accomplish?

After subtracting proj_{u₁}(v₂), what remains is perpendicular to u₁. The subtracted piece was the "shadow" of v₂ along u₁. Normalizing afterwards gives unit length.

Q5. A is 4×6 with singular values 5, 3, 0, 0. What is rank(A)?

Rank = number of nonzero singular values = 2 (σ₁=5 and σ₂=3). The two zeros mean nullity = 4.

Q6. Why do we take the POSITIVE square root when computing σᵢ = √λᵢ from eigenvalues of AᵀA?

Singular values represent how much A stretches vectors. Stretching factors are non-negative by definition. Also, AᵀA is positive semidefinite → eigenvalues ≥ 0 → both roots exist, but we pick positive.

Q7. The shortcut formula uᵢ = (1/σᵢ)Avᵢ fails for which i?

For zero singular values, 1/σᵢ is undefined. The corresponding U columns must be found via the null space of Aᵀ (then apply Gram–Schmidt if needed).

Q8. A is diagonalizable with eigenvalues 2, 2, 3. We know λ=2 is repeated. What MORE do we need to check?

Arithmetic multiplicity of λ=2 is 2. For diagonalizability, geometric multiplicity must also be 2 (i.e., dim of null space of A−2I must be 2). If it's only 1, the matrix is NOT diagonalizable.

Q9. SVD writes A = Σᵢ σᵢuᵢvᵢᵀ. Each term σᵢuᵢvᵢᵀ has rank…

uᵢvᵢᵀ is a column vector times a row vector = outer product. This ALWAYS has rank 1 (one nonzero singular value). A is expressed as the sum of r rank-1 matrices.

Q10. For a symmetric matrix A, how do SVD and eigendecomposition relate?

For symmetric A = QΛQᵀ (spectral theorem), the eigenvectors form both U and V. Singular values = |λᵢ| (absolute values of eigenvalues). Signs of negative eigenvalues are absorbed into U.

Q11. Sum of squares of all singular values of A equals…

Σσᵢ² = eigenvalues of AᵀA summed = trace(AᵀA). Also equals the Frobenius norm squared: ‖A‖_F² = Σᵢⱼ aᵢⱼ². This was the GATE 2024 question!

Q12. Which of these correctly describes what the Gram–Schmidt procedure converts any basis into?

Gram–Schmidt produces orthonormal vectors: first subtract projections (→ orthogonal), then divide by length (→ unit length → orthonormal). This is how SVD's U and V are built.
