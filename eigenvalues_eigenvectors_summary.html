Section 1 What Are Eigenvalues & Eigenvectors?

Eigenvalues and eigenvectors represent a new way of looking at a matrix — distinct from rank. Just as rank is a single number that encodes meaningful information about even a very large matrix, eigenvalues are numbers that carry powerful structural information about a matrix. The central insight is that for any matrix there exist special vectors whose direction does not change when the matrix acts on them — these are the eigenvectors, and the amount by which they are scaled is the eigenvalue.

1.1 Matrix Multiplication as a Geometric Transformation

When a matrix A is multiplied by a vector x, the result is a new vector Ax. Geometrically, this multiplication transforms the original vector — it changes its direction, its length, or both. This is the general behaviour of matrix–vector multiplication.

Illustrative Example — General Transformation

Let A = [[0, 1], [1, 1]] and x = [1, 2]. Then Ax = [2, 3]. The vector [1, 2] is transformed into [2, 3] — its direction has changed. This is the typical case.

There are, however, two distinct possibilities when a matrix acts on a vector:

  • Possibility 1 — Direction changes: The resultant vector Ax points in a different direction from x. This is the most common case for arbitrary vectors.
  • Possibility 2 — Direction preserved: The resultant vector Ax points in exactly the same direction as x, possibly scaled (stretched or shrunk). Vectors for which this second case holds are called eigenvectors.

1.2 Geometric Interpretation — The Ellipse Picture

A particularly illuminating way to understand eigenvectors geometrically is through the transformation of a circle into an ellipse. Consider mapping every point on a unit circle through a matrix A:

  • Every point on the circle gets mapped to a corresponding point — the circle becomes an ellipse.
  • Almost every vector (point on the circle) changes direction after multiplication.
  • Exactly two vectors do not change direction — the vectors lying on the major axis and minor axis of the resulting ellipse. These are the eigenvectors.
  • All other vectors are rotated to new directions; only the eigenvectors are merely scaled.

Concrete Geometric Example

For a specific matrix, the vector [2, 1] transforms to [22, 11] = 11 × [2, 1]. The direction is unchanged; the scaling factor (eigenvalue) is 11. Similarly, the vector [−1, 2] transforms to [−1, 2] — the scaling factor (eigenvalue) is 1. These two vectors are the eigenvectors of that matrix. Every other vector on the circle maps to a point off its original ray direction.

1.3 Formal Definition

Definition — Eigenvalue and Eigenvector

Let A be a square matrix. A non-zero vector v is called an eigenvector of A if there exists a scalar λ such that:

A·v = λ·v

The scalar λ is called the eigenvalue corresponding to eigenvector v.

In words: multiplying A by the eigenvector v produces the same vector v scaled by λ. The direction of v is preserved; only its magnitude changes (by factor λ).

Behaviour based on the value of λ

  • λ = 1: No change — the eigenvector is truly unchanged by A.
  • λ > 1: The eigenvector is stretched — elongated in its own direction.
  • 0 < λ < 1: The eigenvector is shrunk — compressed in its own direction.
  • λ = 0: The eigenvector collapses to the zero vector. This is a valid eigenvalue; it does not violate the definition.
  • λ < 0: The eigenvector is reversed in direction and possibly scaled.

Critical Convention: Eigenvectors Must Be Non-Zero

Definitional Rule

By definition, eigenvectors must be non-zero vectors. The zero vector is explicitly excluded. Reasons:

  • The zero vector has no direction — the geometric concept of "direction preserved" loses all meaning for it.
  • If the zero vector were allowed as an eigenvector, then A·0 = 0 = λ·0 holds for any λ whatsoever — making every scalar an eigenvalue for every matrix, which destroys the usefulness of the definition.

Eigenvalues, however, can be zero. λ = 0 is a perfectly valid eigenvalue as long as the corresponding eigenvector is non-zero.

1.4 Why Eigenvalues Are Defined Only for Square Matrices

The equation Av = λv requires the left side and right side to be vectors of the same dimension. If A is an m×n matrix and v is an n×1 vector, then Av is an m×1 vector. For Av = λv to make sense, the output Av must have the same dimension as v — this requires m = n. Therefore, eigenvalues and eigenvectors are only defined for square matrices.

1.5 Why Eigenvalues Matter — Importance and Applications

Several properties of a matrix can be analysed through its eigenvalues. The lectures emphasise two primary fields of application:

Machine Learning and Regularization

Regularization is a concept used in virtually every machine learning algorithm without exception. Without regularization, models either underfit or overfit. The mathematical core of regularization is built upon eigenvalues and eigenvectors. The dependency chain is:

Machine Learning → Regularization → Eigenvalues & Eigenvectors

This makes eigenvalues foundational to all of modern machine learning.

Principal Component Analysis (PCA)

PCA is an extremely popular dimensionality-reduction technique in machine learning and statistics. The "principal components" — the key directions that capture variance in data — are precisely the eigenvectors of a specific matrix (the covariance matrix). PCA cannot exist without eigenvalue theory.

Section 2 Finding Eigenvalues

2.1 The Characteristic Equation

The Characteristic Equation

The eigenvalues of a matrix A are found by solving:

det(A − λI) = 0

This equation is called the characteristic equation of A. Its solutions (the values of λ that satisfy it) are the eigenvalues.

2.2 Derivation of the Characteristic Equation

Starting from the definition Av = λv (where v is a non-zero eigenvector):

  1. Rearrange: Av − λv = 0, which gives (A − λI)v = 0
  2. This is a homogeneous system of linear equations in v.
  3. For a non-trivial solution to exist (i.e., for v to be non-zero), the matrix (A − λI) must be singular.
  4. A matrix is singular if and only if its determinant is zero.
  5. Therefore: det(A − λI) = 0.

The expression det(A − λI) expands to a polynomial in λ — called the characteristic polynomial. The degree of this polynomial equals the size of the matrix (an n×n matrix gives a degree-n polynomial). Solving this polynomial yields the eigenvalues.

2.3 Properties of Eigenvalues

  • An n×n matrix has exactly n eigenvalues (counting repetitions / multiplicity) — these may be real or complex.
  • For triangular matrices (upper or lower triangular), the eigenvalues are directly the diagonal entries. No computation needed.
  • Eigenvalues can be zero, positive, negative, or complex.
  • The characteristic polynomial of an n×n matrix has degree n, so there are exactly n roots (eigenvalues, possibly repeated).

Worked Example — 2×2 Matrix with Two Distinct Eigenvalues

Let A = [[1, 2], [2, 1]]. Find the eigenvalues.

det(A − λI) = det([[1−λ, 2], [2, 1−λ]]) = (1−λ)² − 4 = 0 = 1 − 2λ + λ² − 4 = 0 = λ² − 2λ − 3 = 0 = (λ − 3)(λ + 1) = 0

Eigenvalues: λ₁ = 3 and λ₂ = −1

Worked Example — 2×2 Matrix with Repeated Eigenvalue

Let A = [[3, 1], [0, 3]]. Find the eigenvalues.

det(A − λI) = (3−λ)² = 0 ⟹ λ = 3 (repeated twice)

Since this is an upper-triangular matrix, the eigenvalue 3 can be read directly from the diagonal. The eigenvalue 3 has arithmetic multiplicity 2.

Worked Example — 3×3 Triangular Matrix

Let A = [[9, ?, ?], [0, 1, ?], [0, 0, 3]] (upper triangular). Eigenvalues are read directly from the diagonal: λ₁ = 9, λ₂ = 1, λ₃ = 3. The characteristic equation can be verified to be (9−λ)(1−λ)(3−λ) = 0 by expanding along the first column.

Section 3 Finding Eigenvectors

3.1 The Process for Finding Eigenvectors

Once the eigenvalues are known, the eigenvector corresponding to each eigenvalue is found by solving a homogeneous system of linear equations. The procedure is as follows:

  1. 1
    Substitute the eigenvalue λᵢ into (A − λᵢI). Subtract λᵢ from each diagonal entry of A to form the matrix B = A − λᵢI.
  2. 2
    Solve B·x = 0. This is a homogeneous system. Apply row reduction (Gaussian elimination) to B.
  3. 3
    Identify free variables. Since det(B) = 0 (by construction — this is how λᵢ was found), B is singular and there will always be at least one free variable.
  4. 4
    Express the solution in terms of free parameters. Let free variables be K, S, T, etc. Express all other variables in terms of these. The resulting vector expression gives the eigenvectors.

Important Distinction

The matrix A and the matrix (A − λI) are completely different matrices and share no special relationship except that λ was chosen to make (A − λI) singular. Do not confuse properties of A with properties of (A − λI).

3.2 Why There Are Always Infinitely Many Eigenvectors

There are always infinitely many eigenvectors corresponding to any given eigenvalue. The reasoning is:

  • If v is an eigenvector of A with eigenvalue λ, then Av = λv.
  • Multiplying both sides by any non-zero scalar k: A(kv) = k(Av) = k(λv) = λ(kv).
  • Therefore kv is also an eigenvector with the same eigenvalue λ, for any non-zero scalar k.
  • Since k can be any non-zero real number, there are infinitely many eigenvectors.

What matters is not the number of eigenvectors (always infinite) but the number of linearly independent eigenvectors — because linearly independent eigenvectors span different directions and carry distinct geometric information.

Algebraically, this infinite set arises from the free variables in the homogeneous system B·x = 0. Each free variable generates a free parameter (K, S, T, ...), and the solution space is spanned by the corresponding basis vectors.

3.3 Comprehensive Worked Examples

Example 1 — Matrix A = [[1,2],[2,1]], Eigenvalues −1 and 3

For λ = −1:

B = A − (−1)I = [[1−(−1), 2],[2, 1−(−1)]] = [[2, 2],[2, 2]] Row reduce: [[2, 2],[0, 0]] Equation: 2x₁ + 2x₂ = 0 → x₁ = −x₂ Let x₂ = K → x₁ = −K Eigenvector: K·[−1, 1] Linearly independent eigenvectors: 1 (just [−1, 1] or [1, −1])

For λ = 3:

B = A − 3I = [[1−3, 2],[2, 1−3]] = [[−2, 2],[2, −2]] Row reduce: [[−2, 2],[0, 0]] Equation: −2x₁ + 2x₂ = 0 → x₁ = x₂ Let x₂ = K → x₁ = K Eigenvector: K·[1, 1] Linearly independent eigenvectors: 1 (just [1, 1])

Example 2 — Matrix with Eigenvalues λ = 1 and λ = 2

A = [[3, −2],[1, 0]]. Characteristic equation: (3−λ)(−λ) + 2 = 0 → λ² − 3λ + 2 = 0 → (λ−1)(λ−2) = 0. Eigenvalues: λ₁ = 1, λ₂ = 2.

For λ = 1: B = [[2, −2],[1, −1]]. Row reduce: [[2, −2],[0, 0]]. Solution: x₁ = x₂ = K. Eigenvector: K·[1, 1].

For λ = 2: B = [[1, −2],[1, −2]]. Row reduce: [[1, −2],[0, 0]]. Solution: x₁ = 2x₂ = 2K. Eigenvector: K·[2, 1].

Example 3 — Identity Matrix

For the 3×3 identity matrix I, every vector x satisfies I·x = x = 1·x. This means every non-zero vector is an eigenvector with eigenvalue 1. There are infinitely many eigenvectors and also infinitely many linearly independent sets among them — specifically, any set of three linearly independent vectors in ℝ³ forms a set of eigenvectors. The identity matrix has one eigenvalue (λ = 1) with full multiplicity (arithmetic multiplicity = 3) and three linearly independent eigenvectors (geometric multiplicity = 3).

Section 4 Linear Independence of Eigenvectors

4.1 Theorem: Eigenvectors from Distinct Eigenvalues Are Linearly Independent

Theorem 1 — Distinct Eigenvalues Imply Linear Independence

If λ₁ and λ₂ are two distinct eigenvalues of a matrix A (i.e., λ₁ ≠ λ₂), then any eigenvector e₁ corresponding to λ₁ and any eigenvector e₂ corresponding to λ₂ are linearly independent.

More generally: eigenvectors corresponding to k pairwise distinct eigenvalues form a linearly independent set of k vectors.

Careful Scope of the Theorem

The theorem speaks about eigenvectors chosen from different eigenvalues. It does not address the linear independence of eigenvectors belonging to the same eigenvalue — those may or may not be linearly independent of each other, depending on the matrix.

For example: corresponding to λ₁, there may be vectors [2,3], [1,2], [4,6] (the last is a scalar multiple of the first). Corresponding to λ₂, there may be vectors [3,5], [2,9], [6,10]. Picking one from each group — say [2,3] and [2,9] — guarantees linear independence since the eigenvalues differ.

4.2 Proof of Theorem 1

The proof uses proof by contradiction.

Given: λ₁ ≠ λ₂. A·e₁ = λ₁·e₁ and A·e₂ = λ₂·e₂.

To prove: e₁ and e₂ are linearly independent.

Assume the contrary: e₁ = k·e₂ for some scalar k (linear dependence).

Step 1: Compute A·e₁ two ways:

Way 1 (direct): A·e₁ = λ₁·e₁ Way 2 (substitute e₁ = k·e₂): A·(k·e₂) = k·(A·e₂) = k·λ₂·e₂

Step 2: Rewrite Way 1 using e₁ = k·e₂:

A·e₁ = λ₁·e₁ = λ₁·k·e₂

Step 3: Equate the two expressions (both equal A·e₁):

λ₁·k·e₂ = k·λ₂·e₂ ⟹ λ₁·k·e₂ − k·λ₂·e₂ = 0 ⟹ k·(λ₁ − λ₂)·e₂ = 0

Step 4: Since e₂ is an eigenvector, e₂ ≠ 0. Therefore, either k = 0 or (λ₁ − λ₂) = 0.

  • If k = 0: then e₁ = 0·e₂ = 0, which contradicts e₁ being an eigenvector (eigenvectors are non-zero).
  • If λ₁ − λ₂ = 0: then λ₁ = λ₂, which contradicts the assumption that λ₁ ≠ λ₂.

Conclusion: Both cases lead to contradictions. Therefore, the assumption that e₁ = k·e₂ is false. Hence e₁ and e₂ are linearly independent. ∎

4.3 The Case of Repeated Eigenvalues

Theorem 1 is silent about what happens when eigenvalues repeat. When an eigenvalue λ appears with multiplicity greater than 1, the number of linearly independent eigenvectors for that eigenvalue can vary. Repeated eigenvalues do not guarantee multiple linearly independent eigenvectors — but they also do not prevent it.

Three concrete examples demonstrate the full range of possibilities when λ is repeated:

Case Eigenvalue Situation Linearly Independent Eigenvectors Found
Example A λ = 3, repeated 3 times 3 (maximum possible)
Example B λ = 1, repeated 3 times 2 (intermediate)
Example C λ = 1, repeated 3 times 1 (minimum possible)

This demonstrates the key insight: when an eigenvalue repeats m times, the number of linearly independent eigenvectors can be anything from 1 to m — inclusive. It cannot exceed m, but it can be any integer between 1 and m.

Example A — λ repeated 3 times, 3 linearly independent eigenvectors

Matrix A = [[1, 0, 0],[0, 1, 0],[0, 0, 1]] (identity, scaled). Characteristic polynomial: (1−λ)³ = 0, so λ = 1 three times. B = A − 1·I = [[0,0,0],[0,0,0],[0,0,0]]. Every vector solves B·x = 0. Free variables: x₁, x₂, x₃ all free. Three linearly independent eigenvectors: [1,0,0], [0,1,0], [0,0,1].

Example B — λ repeated 3 times, 2 linearly independent eigenvectors

A matrix constructed so that (1−λ)³ = 0. After substituting λ = 1, row reduction leaves only one pivot and two free variables (s and t). Eigenvectors: s·[1,1,0] + t·[0,0,1]. Two linearly independent eigenvectors.

Example C — λ repeated 3 times, 1 linearly independent eigenvector

A matrix where (1−λ)³ = 0. After substituting λ = 1, B = [[0,1,0],[0,0,1],[0,0,0]]. Row reduction yields two pivots and only one free variable (x₁ = K, x₂ = 0, x₃ = 0). Only one linearly independent eigenvector: [1, 0, 0].

Section 5 Arithmetic & Geometric Multiplicity

5.1 Definitions

Definition — Arithmetic Multiplicity (AM)

The arithmetic multiplicity of an eigenvalue λ₀ is the number of times λ₀ appears as a root of the characteristic polynomial — equivalently, the power of the factor (λ − λ₀) in the factored form of the characteristic polynomial.

Example: If the characteristic polynomial is (λ − 1)³(λ − 5)¹, then AM(1) = 3 and AM(5) = 1.

Definition — Geometric Multiplicity (GM)

The geometric multiplicity of an eigenvalue λ₀ is the number of linearly independent eigenvectors corresponding to λ₀ — equivalently, the number of free variables when solving (A − λ₀I)·x = 0, equivalently, the dimension of the null space of (A − λ₀I), equivalently, n − rank(A − λ₀I) for an n×n matrix.

5.2 The Fundamental Inequality and Its Implications

Fundamental Inequality

For any eigenvalue λ₀ of any square matrix A:

1 ≤ GM(λ₀) ≤ AM(λ₀)

The geometric multiplicity is always at least 1 (every eigenvalue has at least one eigenvector) and at most equal to the arithmetic multiplicity.

Reading Bounds from the Characteristic Equation

Given a characteristic equation such as (λ−λ₁)^m₁ · (λ−λ₂)^m₂ · ... · (λ−λₖ)^mₖ = 0, we can immediately state:

  • Minimum (at least) linearly independent eigenvectors: k (one for each distinct eigenvalue, guaranteed by Theorem 1)
  • Maximum (at most) linearly independent eigenvectors: m₁ + m₂ + ... + mₖ = n (the total degree of the polynomial = dimension of A)

Applying the Bounds — Characteristic Equation (λ−1)²(λ−5)¹(λ−7)³ = 0

Matrix dimension: 2 + 1 + 3 = 6 → A is 6×6.

Distinct eigenvalues: 3 (λ=1, λ=5, λ=7) → at least 3 linearly independent eigenvectors.

Total degree: 6 → at most 6 linearly independent eigenvectors.

Correspondence to individual eigenvalues: AM(1) = 2 → GM(1) ∈ {1, 2}; AM(5) = 1 → GM(5) = 1; AM(7) = 3 → GM(7) ∈ {1, 2, 3}.

Correct answers: "at least 3" (option D) and "at most 6" (option B). Both B and D are correct in the MSQ formulation.

5.3 Three Worked Examples Demonstrating AM vs. GM Variation

Example Set — Three 3×3 Matrices, All with λ = 1 (AM = 3), Different GM

Matrix 1: A₁ = [[1,0,0],[0,1,0],[0,0,1]] (identity). B = 0. Three free variables. GM(1) = 3. AM = GM. ✓

Matrix 2: A₂ (constructed). After substituting λ = 1, B has rank 1 and two free variables. GM(1) = 2. AM > GM.

Matrix 3: A₃ = [[1,1,0],[0,1,1],[0,0,1]] (Jordan block). After substituting λ = 1, B = [[0,1,0],[0,0,1],[0,0,0]] has rank 2 and one free variable. GM(1) = 1. AM > GM significantly.

These three matrices have the same eigenvalue (λ=1, AM=3) but GM = 3, 2, 1 respectively — demonstrating the full range.

Key Conclusion

The power (AM) of an eigenvalue in the characteristic polynomial gives an upper bound but not an exact count of linearly independent eigenvectors. Only in special cases (particularly real symmetric matrices — see Section 6) is AM = GM guaranteed.

5.4 Reasoning from Characteristic Equations — Advanced Problems

Forward Problem: From Characteristic Equation to Number of Eigenvectors

Given the factored characteristic equation, determine bounds on the number of linearly independent eigenvectors.

Example: (λ−1)³(λ−5)⁶(λ−10)^4(λ)^5 = 0

Dimension: 3+6+4+5 = 18 → 18×18 matrix.

Distinct eigenvalues: 4 (λ=1, λ=5, λ=10, λ=0). At least 4 linearly independent eigenvectors.

At most 18 linearly independent eigenvectors.

For λ=0 specifically: GM(0) ∈ {1, 2, 3, 4, 5} — up to 5 linearly independent eigenvectors.

Reverse Problem: From Number of Free Variables to AM

Example: A is 20×20. Solving (A−3I)x = 0 yields 7 free variables. What is the power of (λ−3) in the characteristic equation?

7 free variables → GM(3) = 7 → AM(3) ≥ 7. So the power of (λ−3) is at least 7. It cannot be less than 7 (AM ≥ GM). It could be 7, 8, 9, ..., up to 20 (the matrix dimension). Correct options among choices: 7 and 20.

Problem: Only 2 Linearly Independent Eigenvectors in a 5×5 Matrix

If a 5×5 matrix has only 2 linearly independent eigenvectors total, then:

  • At most 2 distinct eigenvalues are possible (since 3 distinct eigenvalues would guarantee at least 3 linearly independent eigenvectors by Theorem 1).
  • Characteristic equations with 3 or more distinct eigenvalue factors are not possible.
  • Valid forms include: (λ−λ₁)^5 alone (if only one eigenvalue), or (λ−λ₁)^a(λ−λ₂)^b with a+b=5 (if two eigenvalues, each contributing exactly 1 linearly independent eigenvector).
  • This situation necessarily implies repeated eigenvalues exist since 5 linearly independent eigenvectors would require 5 distinct eigenvalues, but we only have 2.

Using Rank to Find Geometric Multiplicity

Formula

GM(λ₀) = n − rank(A − λ₀I)

If A is 5×5 and rank(A − λ₁I) = 2, then GM(λ₁) = 5 − 2 = 3. There are 3 linearly independent eigenvectors for λ₁.

Section 6 Real Symmetric Matrices — Special Properties

Real symmetric matrices occupy a privileged position in linear algebra. They have two fundamental properties that are not shared by general square matrices.

6.1 Two Key Properties of Real Symmetric Matrices

Theorem 2 — Properties of Real Symmetric Matrices

For any real symmetric matrix A (i.e., A = Aᵀ with all real entries):

  • Property 1 — All eigenvalues are real. No complex eigenvalues exist, even though general matrices can have complex eigenvalues.
  • Property 2 — All eigenvectors are orthogonal. Eigenvectors corresponding to distinct eigenvalues are perpendicular to each other. Moreover, even for repeated eigenvalues, one can always choose eigenvectors to be mutually orthogonal (via Gram–Schmidt).

Orthogonal vectors are also linearly independent. Therefore, all eigenvectors of a real symmetric matrix are linearly independent.

6.2 AM = GM Is Guaranteed for Real Symmetric Matrices

Theorem 3 — AM Equals GM for Real Symmetric Matrices

For a real symmetric matrix A, the arithmetic multiplicity equals the geometric multiplicity for every eigenvalue. Concretely:

  • If A is n×n, then A has exactly n linearly independent eigenvectors.
  • For each eigenvalue, the number of linearly independent eigenvectors equals its power in the characteristic polynomial — no more, no less.
  • This holds even when eigenvalues are repeated.

Example — Real Symmetric Matrix, Characteristic Equation (λ−1)³(λ−5)⁷(λ−11)¹ = 0

Since A is real symmetric: AM = GM for every eigenvalue.

  • λ = 1: AM = 3 → exactly 3 linearly independent eigenvectors.
  • λ = 5: AM = 7 → exactly 7 linearly independent eigenvectors.
  • λ = 11: AM = 1 → exactly 1 linearly independent eigenvector.
  • Total: 3 + 7 + 1 = 11 linearly independent eigenvectors. (The matrix is 11×11.)

For a general (non-symmetric) matrix with the same characteristic equation, the count would be "at least 3, at most 11."

6.3 Connection to Diagonalization

The property AM = GM for all eigenvalues is precisely the condition needed for a matrix to be diagonalizable. A matrix is diagonalizable if and only if it has n linearly independent eigenvectors (where n is its dimension).

Consequences for real symmetric matrices:

  • Real symmetric matrices are always diagonalizable.
  • They can be written as A = PΛP⁻¹ where P has eigenvectors as columns and Λ is diagonal with eigenvalues on the diagonal.
  • Since eigenvectors are orthogonal, P is an orthogonal matrix (P⁻¹ = Pᵀ), giving the Spectral Theorem: A = PΛPᵀ.
  • Diagonalization enables many algorithms in machine learning, numerical methods, and applied mathematics to work efficiently.

Summary of When AM = GM is Guaranteed

The equation AM = GM (and thus guaranteed diagonalizability) holds for:

  • Any matrix where all eigenvalues are distinct (Theorem 1 applies; GM = AM = 1 for each).
  • All real symmetric matrices (Theorem 2 applies; GM = AM for all eigenvalues, even repeated ones).

For a general matrix with repeated eigenvalues, AM = GM is not guaranteed — it must be verified by computing GM directly.

Section 7 Applications & Significance

7.1 Central Importance in Machine Learning

Eigenvalues and eigenvectors are not a peripheral topic in machine learning — they are embedded at the mathematical foundations of the discipline. The lectures establish the following dependency:

Deep Learning / Machine Learning ↓ (cannot function without) Regularization ↓ (mathematical core is) Eigenvalues & Eigenvectors

This transitivity makes eigenvalue theory as fundamental to machine learning as calculus or probability theory.

7.2 Regularization

Regularization is a technique applied in virtually every machine learning algorithm to control model complexity, prevent overfitting (model too closely fitted to training data), and prevent underfitting (model too simple to capture patterns). Without regularization:

  • Models fit the training data perfectly but generalise poorly to new data (overfitting).
  • Or models are too simple to capture the structure of the data (underfitting).

The mathematical mechanism behind regularization — how it penalises model complexity — is directly expressed in terms of eigenvalues and eigenvectors of the relevant matrices. This connection is left at an intuitive level in the lectures but is flagged as critical for deeper mathematical study at the Master's level.

7.3 Principal Component Analysis (PCA)

Principal Component Analysis (PCA) is described as an "extremely popular" application built directly on eigenvector theory. In PCA:

  • A dataset is represented as a matrix.
  • The covariance matrix (or a related matrix) is computed — it is always a real symmetric matrix.
  • The eigenvectors of this covariance matrix are the principal components — the directions in the data that capture the most variance.
  • Projecting data onto the top k eigenvectors (those with the largest eigenvalues) gives a low-dimensional representation that preserves as much information as possible.

PCA is used in dimensionality reduction, data visualisation, noise reduction, and feature extraction across virtually all domains of machine learning and data science.

8.1 Complete Procedure for Finding Eigenvalues and Eigenvectors

  1. 1
    Set up the characteristic equation: Compute det(A − λI) = 0.
  2. 2
    Solve for λ: Expand the determinant to get the characteristic polynomial. Factor and find roots. These roots are the eigenvalues.
  3. 3
    For each eigenvalue λᵢ: Form B = A − λᵢI and row-reduce B.
  4. 4
    Identify free variables in the reduced system and express the solution in parametric form.
  5. 5
    Write the eigenvectors as span of the basis vectors arising from free parameters.

Found by solving det(A − λI) = 0 (characteristic equation). → Can be zero, positive, negative, or complex. → Number of eigenvalues = dimension of matrix (counting AM). → For triangular matrices: read directly from diagonal. → Governs stretching/shrinking behaviour: |λ|>1 stretches, |λ|<1 shrinks, λ=0 collapses. → Determines diagonalizability through AM vs. GM comparison.
Characteristic Equation / Polynomial
det(A − λI) = 0. Degree-n polynomial for n×n matrix. → Roots are eigenvalues. → Factored form (λ−λ₁)^m₁···(λ−λₖ)^mₖ encodes arithmetic multiplicities. → Sum of all powers = n (matrix dimension).
Arithmetic Multiplicity (AM)
Power of (λ−λ₀) in characteristic polynomial. → Upper bound for GM: AM ≥ GM. → Equals GM for real symmetric matrices (Theorem 3). → Determines the maximum possible number of linearly independent eigenvectors for a given λ.
Geometric Multiplicity (GM)
Number of linearly independent eigenvectors for a given λ. Equals dim(null space of A−λI) = n − rank(A−λI). → Always ≥ 1 and ≤ AM. → Determined by number of free variables in (A−λI)x = 0. → Equals AM for real symmetric matrices. → Sum of all GMs = total linearly independent eigenvectors = n (if diagonalizable).
Theorem 1: Distinct λ → Lin. Ind. Eigenvectors
If λ₁ ≠ λ₂, their eigenvectors are linearly independent. → Proved by contradiction. → Guarantees at least k linearly independent eigenvectors if k distinct eigenvalues exist. → Sufficient but not necessary for diagonalizability. → Connects to diagonalizability when all eigenvalues are distinct.
Theorem 2: Real Symmetric → Real λ, Orthogonal Eigenvectors
For real symmetric A: all λ real; all eigenvectors orthogonal (and thus linearly independent). → Enables Spectral Theorem: A = PΛPᵀ. → Guarantees diagonalizability. → Enables PCA and other eigendecomposition-based algorithms. → Underpins regularization in machine learning.
Theorem 3: AM = GM for Real Symmetric Matrices
Strongest guarantee: even repeated eigenvalues produce exactly AM-many linearly independent eigenvectors. → Means the matrix always has n linearly independent eigenvectors. → Implies diagonalizability. → Key difference from general matrices where GM can be strictly less than AM.
Diagonalizability
A matrix is diagonalizable iff it has n linearly independent eigenvectors (sum of all GMs = n). → Requires GM = AM for all eigenvalues simultaneously. → Always holds for: (a) matrices with n distinct eigenvalues; (b) real symmetric matrices. → Enables efficient computation of matrix powers: Aᵏ = PΛᵏP⁻¹. → Foundational for SVD, PCA, and other matrix factorizations.
Linear Independence of Eigenvectors
Central concept connecting eigenvalue theory to linear algebra foundations. → Determined by counting free variables in (A−λI)x = 0. → Guaranteed across different λ values (Theorem 1). → Guaranteed within same λ for real symmetric matrices (Theorem 2&3). → Not guaranteed for repeated λ in general matrices. → Necessary condition for diagonalizability.
Rank of (A − λI)
rank(A − λI) = n − GM(λ). Used to compute GM. → Since det(A − λI) = 0 for eigenvalues, rank < n always. → Higher rank → fewer linearly independent eigenvectors (smaller GM). → Connects null space theory to eigenvalue theory.
Applications: PCA, Regularization, Diagonalization
All three depend on eigenvector theory. → PCA: principal components = eigenvectors of covariance matrix (real symmetric). → Regularization: mathematical core = eigenvalue analysis. → Diagonalization: enables matrix powers, differential equations, Google's PageRank, etc. → Chain: ML → Regularization → Eigenvalues/Eigenvectors.
